Lesson 1: Use Functions

Abstraction

Abstraction allows us to focus on the actual program we want to create while utilizing functions which are included in the software langauge, examples include opening a browser or printing the time. E.g.

import time
time.sleep(10)

This example shows how the time function is brought into the coding and used. In this example the code will be run after a ten second delay.

Lesson 2: Use Classes

Classes and Instances (or Objects)

Classes are like blueprints of a building containing the dimensions. An instance is the creation of a new object using the same class which may be altered in some way, such as changing the color of the building.

Questions about Answers

What is an OOP language?

An orient-oriented programming language allows sturcturing to create consistent, clear and simplified code.

What is a class?

A class is a template for creating objects in code. They are the blueprints which can then be adapted for each object which needs to be created.

What is an instance?

An instance is an single example of a class.

For example vehicles may be a class. Cars may be another example of a class which may inherit some of the information from the vehicle class. A specific model of a car may be an instance.

Lesson 3a: Making Classes

Constructor

Init means "initialized", this reserves space in memory to store the details information for the relevant object.

class Movie():
 def __init__(self, movie_title, movie_storyline)
  self.title = movie_title (this is the first instance variable)
  self.storyline = movie_storyline (this is the second instance variable)

toy_story = Movie("Toy Story","A story of a boy and his toys that come to life") (this is an instance)

print(toy_story.title) --> Toy Story
print(toy_story.storyline) --> A story of a boy and his toys that come to life.

Note: The class and the instance would normally be created in two separate files stored in the same folder. To use the class the simply typing import and the class name at the top of the file containing the instance, e.g.import Movie would enable the class Movie to be used. The convention is to name classes with a capital letter.

Lesson 3b: Advanced Topic

Inheritance

Inheritance allows variables from a class to be used in a second class. There may be a class "Parent" which contains the variable "surname" and "city of birth". A "Child" class may inherit these variable in addition to its own variable of "school".

class Parent():
 def __init__(self, last_name, city_of_birth):
  self.last_name = last_name
  self.city_of_birth = city_of_birth

class Child(Parent):
 def __init__(self, last_name, city_of_birth, school):
  Parent.__init__(self, last_name, city_of_birth)
  self.school = school

george = Child("Bush", "New Haven", "Yale")
print george.last_name --> Bush
print george.city_of_birth --> New Haven
print george.school --> Yale

Method Overriding

If there is a function or variable with the same name appears in the parent and the child the child takes precidence; this is known as method overriding.
